Détail des modifications

Frontend — composants générateur

  • Step1_ThemeSelection.tsx : ajout du helper isCalculated(), filtrage des datasets tool:"Calcul", retrait du disabled={!demoReady} (tous cliquables), nouveau badge « Import requis » au lieu de « Bientôt », clé React composite subThemeId::id::variable pour les boutons dupliqués.
  • GeneratorPage.tsx : ajout de selectedDatasetVariable en state + sessionStorage, propagation via handleDatasetSelect(themeId, subThemeId, datasetId, variable). findDataset matche désormais sur (id, variable) → règle le bug breadcrumb suicide (dataset comp_mortalite partagé entre alcool/tabac/suicide).
  • SidebarSummary.tsx : mêmes ajustements sur findDatasetLabel, ajout de la prop selectedDatasetVariable.

Backend

  • themes_config.json : 3 nouveaux datasets — route (nb_acci, tx_acci, nb_blesses, tx_blesses, nb_morts, tx_morts), noyades (nb_noyades, tx_noyades, nb_noyades_deces, tx_noyades_deces), comp_mortalite (m_alcool, m_tabac, m_suicide). Structure alignée sur accueil_pop_inf3ans (parser external).
  • set_all_admin.py : script idempotent, auth _superusers puis fallback admins, pagination 200/page de la collection users, PATCH role=admin avec skip si déjà admin.
